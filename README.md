# ViteJs Tailwind Starter Kit


This is a starter kit for building web applications using ViteJS and Tailwind CSS. The project is designed to get you up and running quickly with a modern web development workflow that leverages the power of these two popular tools.

## Getting Started

To get started, follow these simple steps:

1. Clone the repository: 

```
git clone https://github.com/ashuydv/vite-tailwind-starter-kit/ 
```

2. Navigate to the project directory: 

```
cd vite-tailwind-starter-kit
```

3. Install the dependencies: 

```
npm install
```

4. Start the development server:

```
npm run dev
```

5. Open your browser and navigate to http://localhost:3000. You should see the default starter page. 


## Features 

This starter kit comes with the following features out of the box: 

* ViteJS for fast and efficient development.
* Tailwind CSS for rapid UI prototyping and styling.
* Router for navigating through files.

## Usage

This starter kit is intended to be used as a starting point for building web applications using ViteJS and Tailwind CSS. Feel free to modify the code and configuration to suit your specific needs.
